# 세계관 설정 노트

AI 채팅용 세계관을 정리하는 개인용 웹페이지야.

## 쓰는 방법

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있어.

입력한 내용은 현재 브라우저의 저장 공간에 자동 저장돼.

## 들어있는 기능

- 세계관 개요 정리
- 조직도 추가와 수정
- 캐릭터 카드 추가와 수정
- 사건 기록 추가와 수정
- 자유 설정 메모
- GitHub에 저장하고 다른 PC에서 불러오기

## 웹에 올리는 방법

GitHub 저장소에 올린 뒤 GitHub Pages를 켜면 다른 PC에서도 볼 수 있어.

GitHub Pages 설정:

1. GitHub 저장소로 이동
2. Settings
3. Pages
4. Branch를 `main`으로 선택
5. 폴더는 `/root` 선택
6. Save

잠시 기다리면 GitHub Pages 주소가 생겨.

## 다른 PC에서도 같은 내용 쓰기

웹페이지 안의 `클라우드` 메뉴를 사용하면 돼.

1. GitHub 토큰을 넣기
2. `GitHub에 저장` 누르기
3. 다른 PC에서 같은 주소 열기
4. 같은 토큰을 넣고 `GitHub에서 불러오기` 누르기

저장되는 파일은 기본으로 `world-data.json`이야.

토큰은 공개 파일에 저장하지 않아. 네 브라우저 안에만 저장돼.
